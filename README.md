# CodeAlpha Data Science Internship

## Task 1: Iris Flower Classification

### Internship Details
This project was completed as part of the CodeAlpha Data Science Internship Program.

### Project Overview
The objective of this project is to build a Machine Learning model that can classify iris flowers into three different species:

- Iris Setosa
- Iris Versicolor
- Iris Virginica

The classification is based on the following flower measurements:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

### Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

### Project Steps
1. Imported the Iris dataset.
2. Performed data preprocessing and analysis.
3. Split the dataset into training and testing sets.
4. Trained a Machine Learning classification model.
5. Evaluated the model performance.
6. Predicted flower species using the trained model.

### Files Included
- iris.csv – Dataset
- iris_task1.ipynb – Jupyter Notebook
- REQUIREMENTS.txt – Required Python libraries
- Video_Iris.mp4 – Project demonstration video
- README.md – Project documentation

### Outcome
Successfully developed a Machine Learning model capable of classifying iris flower species with high accuracy.
